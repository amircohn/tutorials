## Tips
#### Autosave
- settings (icon) -> settings - "user settings" 
```
"files.autoSave": "onFocusChange"
```

#### git
- git ignore

### Keybindings
- keybindings.json
- add dictionary item to list
```
[
    {
        "key": "ctrl+d",
        "command": "editor.action.deleteLines",
        "when": "editorTextFocus"
    }
]
```
